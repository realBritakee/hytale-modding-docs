## Authentication Commands

### auth login device

Authenticates your server with Hytale. This command prints a URL that you can use to authenticate the server using your Hytale account.

```
auth login device
```

### auth persistence Encrypted

Keeps your server authenticated after restarting. Run this after authenticating once.

```
auth persistence Encrypted
```

**Warning:** Never share your encrypted auth file!
